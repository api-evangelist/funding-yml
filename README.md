# FUNDING.yml (funding-yml)
GitHub configuration file specifying funding platforms and links for project sponsorship, displayed as a Sponsor button on the repository page. Supports platforms including GitHub Sponsors, Patreon, Open Collective, Ko-fi, Liberapay, Tidelift, Buy Me a Coffee, IssueHunt, Polar, thanks.dev, LFX Mentorship, and custom URLs.
